# EdgeAI Labeling

EdgeAI Labeling is a lightweight data annotation tool designed for computer vision tasks such as vehicle and license plate detection.

Built for real-world AI systems like smart parking, it supports fast labeling, YOLO format export, and AI-assisted auto annotation.

## Features
- Bounding box labeling (multi-object)
- YOLO format export
- Auto labeling using YOLOv8
- Image navigation (next/prev)
- Clean and simple UI

## Use Cases
- AI Parking
- Vehicle detection
- License plate recognition
- Custom dataset labeling
